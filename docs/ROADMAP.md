# Roadmap

## Fase 1 (MVP atual)

- painel admin único
- temas prontos
- exportação HTML/PDF

## Fase 2

- modelagem por bloco (capa, sumário, receitas, dicas, autor)
- validação de campos obrigatórios
- presets visuais adicionais

## Fase 3

- parser DOCX (PHPWord)
- parser PDF texto (smalot/pdfparser)
- matching de imagem por título de receita

## Fase 4

- QA automático (ortografia, estrutura e contagem)
- histórico de versões do ebook
- comparação entre versões no admin

## Fase 5

- geração de imagens via API (opcional)
- fluxo de aprovação/publicação
- biblioteca de templates premium
