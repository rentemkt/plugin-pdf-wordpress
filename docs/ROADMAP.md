# Roadmap

## Implementado (até v0.4.1)

- painel admin único com navegação por seções
- gestão de projetos/clientes
- editor estruturado + modo bruto sincronizado
- categorias com subcapa (nome, subtítulo e imagem)
- importação híbrida (upload + Google Drive com subpastas)
- auditoria de importação no painel
- prévia HTML local e prévia PDF fiel
- exportação HTML/PDF
- laboratório de transcrição (áudio e vídeo)
- saída de transcrição em TXT/SRT/VTT/Lipsync
- processamento de mídia longa com segmentação e merge
- retomada de transcrição por `resume_token` e botão `Retomar da parte N`

## Próxima fase (prioridade alta)

- validação final automática de conteúdo antes do PDF (campos essenciais por item)
- painel de comparação entre versões de projeto
- melhoria visual do editor para operação diária (mais compacto em telas menores)
- suíte mínima de testes automatizados para regressão dos fluxos de importação/transcrição

## Fase seguinte

- QA de texto (ortografia/gramática pt-BR) com sugestões aplicáveis
- histórico de versões com diff por item
- presets visuais adicionais para capas e páginas internas

## Fase futura

- geração assistida de imagens por API (opcional)
- fluxo de aprovação/publicação
- biblioteca de templates premium
